# 🏠 Smart Home IoT Dashboard

Este projeto é uma simulação de um sistema de casa inteligente (Smart Home IoT), desenvolvido com HTML, CSS e JavaScript.

## 🎯 Objetivo
Criar uma interface web interativa para simular o controlo de dispositivos IoT de uma residência moderna.

## 💡 Funcionalidades

- 💡 Controlo de Smart LEDs (ligar/desligar e brilho)
- ❄️ Controlo de ar-condicionado
- 🌡️ Termostato simulado
- 🚨 Detector de fumaça e monóxido de carbono
- 🏃 Sensor de movimento
- 🔔 Sistema de alarme
- 💦 Sistema sprinkler (emergência)
- 🚪 Porta e janela inteligentes
- 📷 Câmera de segurança simulada

## 🖥️ Tecnologias usadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🌐 Demonstração
Este projeto pode ser hospedado com GitHub Pages para funcionar como site online.

- Melhorar UI/UX
- Adicionar backend e comunicação em tempo real
